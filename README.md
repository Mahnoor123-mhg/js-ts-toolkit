# JS TS Toolkit

This project is a small utility toolkit created using Modern JavaScript and TypeScript.

## Objective

The objective of this assignment is to build utility functions in JavaScript and convert them into TypeScript with proper types, interfaces, generics, and return types.

## Utilities Included

1. Deep clone an object
2. Debounce function
3. Fetch wrapper with error handling
4. Array groupBy function

## Modern JavaScript Concepts Used

- Destructuring
- Rest and spread operators
- Optional chaining
- Nullish coalescing
- Async/await
- Closures

## TypeScript Concepts Used

- Interfaces
- Generics
- Proper return types
- Type safety
